# react_router
React-routerを使用して宣言的記法を利用したSPAのチュートリアル

## 使い方
ターミナルを開いて
```
npm install
```
でnode_modulesを作成
```
npm start
```
でサーバーを開いてRun

## 環境
Mac OS  
VS Code
