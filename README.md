# react_router
React-routerを使用して宣言的記法を利用したSPAのチュートリアル

## 使い方
ターミナルを開いて
```
$ npm install --save-dev @babel/core @babel/preset-env @babel/preset-react babel-loader \
        webpack webpack-cli webpack-dev-server \
        react react-dom \
        react-router react-router-dom
```
で必要なライブラリのインストール
```
npm start
```
でサーバーを開いてRun

## 環境
Mac OS  
VS Code
